## 📖 Overview

RealEstate3.0 enables fractional ownership of real estate through blockchain tokenization. Built on Ethereum with React frontend.

### Key Features
- 🏠 Tokenize Properties (ERC-721 NFTs)
- 💰 Fractional Sales (ERC-20 Tokens)
- 🛒 Buy/Sell Fractions 24/7
- 📊 Track Portfolio
- 🌍 Global Access
